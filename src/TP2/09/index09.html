Pergunta: Qual das opções abaixo é uma prática recomendada para garantir a responsividade de um site?

  • A - Definir todas as larguras em pixels para ter controle sobre o layout.
  • B - Usar imagens de boa qualidade sem otimização para garantir a qualidade visual.
  • C - Aplicar media queries para ajustar o design em diferentes tamanhos de tela.
  • D - Bloquear o zoom do usuário para manter o layout.
  • E - Usar apenas fontes grandes e fixas para facilitar a leitura em todas as telas.

Resposta Correta: C - Aplicar media queries para ajustar o design em diferentes tamanhos de tela.

Justificativa: O uso de media queries permite que o design de um site seja adaptado a diferentes dispositivos e tamanhos de tela, proporcionando uma melhor experiência para todos os usuários, independentemente do dispositivo que eles estejam usando.
