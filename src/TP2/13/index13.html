1. O que é Mobile-first?

A -Fazer o site apenas para dispositivos móveis.
B -Fazer o site com base em dispositivos móveis e depois adaptar para dispositivos maiores.
C -Fazer o site exclusivamente para tablets.
D - Criar um site que se ajuste automaticamente à resolução da tela.

Resposta correta: B

Justificativa: O conceito Mobile-first refere-se a desenvolver um site com base em dispositivos móveis e, em seguida, adaptar para telas maiores, garantindo uma boa experiência em dispositivos pequenos antes de expandir para dispositivos maiores.

2. Qual é a principal vantagem de usar a responsividade em um site?

A - Aumentar a quantidade de anúncios exibidos.
B - Melhorar a experiência do usuário em diferentes tamanhos de tela.
C - Reduzir a necessidade de testes de usabilidade.
D - Tornar o site mais rápido em todos os dispositivos.

Resposta correta: B

Justificativa: A principal vantagem de utilizar técnicas de responsividade é melhorar a experiência do usuário em diferentes tamanhos de tela, garantindo que o conteúdo seja acessível e legível em qualquer dispositivo.

3. O que é uma media query em CSS?

A - Uma regra CSS que aplica estilos com base no tamanho da tela.
B - Uma ferramenta para aumentar a velocidade de carregamento.
C - Um método para criar layouts fixos.
D - Um tipo de animação CSS.

Resposta correta: A

Justificativa: Media queries são regras CSS que permitem aplicar estilos diferentes com base no tamanho da tela e outras características do dispositivo, possibilitando a criação de designs responsivos.

4. Qual é o jeito recomendado para otimizar um site para dispositivos móveis?

A - Usar imagens de alta resolução.
B - Usar tamanhos de fonte grandes e um layout simplificado.
C - Evitar o uso de estilos CSS.
D - Criar um design fixo sem alterações para diferentes dispositivos.

Resposta correta: B

Justificativa: Para otimizar um site para dispositivos móveis, é recomendável usar tamanhos de fonte adequados e um layout simplificado que facilite a navegação e a leitura em telas menores.

5. O que é o conceito de design fluido?

A - Um design que se adapta ao tamanho da tela utilizando unidades de medida relativas.
B - Um design que mantém um layout fixo independentemente da resolução da tela.
C - Um design que utiliza apenas imagens estáticas.
D - Um design que não requer qualquer tipo de ajuste para diferentes dispositivos.

Resposta correta: A

Justificativa: O design fluido utiliza unidades de medida relativas, como porcentagens, para que o layout se ajuste suavemente ao tamanho da tela, proporcionando uma experiência consistente em diferentes dispositivos.
